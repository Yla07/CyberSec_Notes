It is an essential server that is full of qualified domain names and their corresponding [IPv4 or IPv6](IP.md) addresses. It has various types of records including following:

- ***A record*** - is used to map domain name to and IPv4 address.

- ***AAAA record***  - maps domain name to IPv6 address.

- ***Canonical name record*** - (CNAME record) works as an alias and maps one domain to another.

- ***Mail exchanger record*** -  (MX record)lists mail server for domain male exchange.

- ***Pointer record*** - (PTR record) is  reverse lookup that matches the IP address to domain name
