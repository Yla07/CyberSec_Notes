When a user enters a website domain and tries to access it, HTTP establishes a connection to the server hosting the domain and provides access to the website. 
For example, when a user types a domain name, such as *google.com*, into their browser, HTTP connects to the web server hosting that domain. 
The web server then responds by sending the HTML content or the code that defines the structure and content of the web page.

A Form of HTTP request is HTTPS (HTTP Secure) it can encrypt user data providing  a more secure connection.
