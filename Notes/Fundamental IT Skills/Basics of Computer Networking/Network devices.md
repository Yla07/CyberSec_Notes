- Router - can connect multiple network like home  network with the internet. It's job is direct data packets to their correct destination.

- Switch - Connects devices within the same network and directs data packets to the specific  device it is meant for.

- Hub - basic device that connects devices in the same network. Unlike switch it sends data to all devices which is less efficient than switch. 

- Bridge - connects and filters  traffic between two networks or segments. Helps to reduce network traffic.

- Gateway - Connects two different types of networks and translates data between different protocols.

- Access Point (AP) - provide s wireless connection to devices. extends a wired network into a  WiFi network

- Modem - Converts digital data from computers into signals for phone/cable and the other way around.Provides internet access.

- Firewall - Monitors and controls incoming and outgoing network traffic and provides security by blocking unauthorized access.
